# Quiz Battle Platform

Quiz game made with HTML, CSS, JavaScript.

## How to Run
1. Open index.html in browser
2. Enter name, select category
3. Click Start Quiz

## Features
- 5 Categories: HTML, CSS, JS, Python, General
- 30 second timer per question
- Score tracking +10 per correct
- Leaderboard with LocalStorage
- Responsive for mobile

## Tech Used
HTML5, CSS3, Vanilla JavaScript
